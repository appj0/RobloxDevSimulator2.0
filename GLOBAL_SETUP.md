# Global Multiplayer Data Setup (Free with Supabase)

This game now supports a global leaderboard backend.

## 1. Create a free Supabase project

1. Go to `https://supabase.com/`
2. Create a new project (free plan).

## 2. Create leaderboard table

Run this SQL in Supabase SQL Editor:

```sql
create table if not exists public.leaderboard_runs (
  id bigint generated by default as identity primary key,
  "studioName" text not null,
  "score" numeric not null default 0,
  "cash" numeric not null default 0,
  "net" numeric not null default 0,
  "visits" numeric not null default 0,
  "peak" numeric not null default 0,
  "games" integer not null default 0,
  "activeGames" integer not null default 0,
  "deadGames" integer not null default 0,
  "avgScore" numeric not null default 0,
  "updates" integer not null default 0,
  "totalPayout" numeric not null default 0,
  "debt" numeric not null default 0,
  "reputation" numeric not null default 0,
  "creditScore" numeric not null default 0,
  "teamSize" integer not null default 0,
  "revenue" numeric not null default 0,
  "expense" numeric not null default 0,
  "bestGameTitle" text not null default 'N/A',
  "bestGameScore" numeric not null default 0,
  "bestGamePeak" numeric not null default 0,
  "bestGameVisits" numeric not null default 0,
  "runtime" text not null default '0y 0m 0d',
  "runtimeDays" integer not null default 0,
  "source" text not null default 'retired',
  "createdAt" bigint not null
);
```

## 3. Allow public read + insert

Enable RLS and policies:

```sql
alter table public.leaderboard_runs enable row level security;

create policy "public read leaderboard_runs"
on public.leaderboard_runs
for select
to anon
using (true);

create policy "public insert leaderboard_runs"
on public.leaderboard_runs
for insert
to anon
with check (true);
```

## 4. Fill `config.js`

Edit `game2/config.js`:

- `provider`: `"supabase"`
- `supabaseUrl`: your project URL
- `supabaseAnonKey`: your anon/public key
- `table`: `"leaderboard_runs"`

## 5. Deploy

Publish your site (e.g. GitHub Pages).  
Once configured, disbanded runs are pushed globally and leaderboard page pulls global records.
